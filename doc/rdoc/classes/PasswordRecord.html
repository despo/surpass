Class PasswordRecord
In: lib/biff_record.rb
Parent: BiffRecord

This record is part of the worksheet/workbook protection. It stores a 16-bit hash value, calculated from the worksheet or workbook protection password.

     # File lib/biff_record.rb, line 166
166:   def password_hash(plaintext)
167:     return 0 if plaintext === ""
168:     hash = 0x0000
169:     plaintext.unpack('C*').each_with_index do |t, i|
170:       c = t << (i + 1)
171:       low_15 = c & 0x7fff
172:       high_15 = c & 0x7fff << 15
173:       high_15 = high_15 >> 15
174:       c = low_15 | high_15
175:       hash = (hash ^ c)
176:     end
177:     hash = (hash ^ plaintext.length)
178:     hash = (hash ^ 0xCE4B)
179:     hash
180:   end
